# WynHealth Rover 
A HAPE (Health and Physical Education) community health assignment project. This repository contains the source code for a promotional portfolio website designing a fictional mobile health program for young people in Wyndham, Victoria.

## Project Overview

**WynHealth Rover** is a conceptual mobile health van program designed to address documented gaps in youth-focused health services in Wyndham. The program proposes free 15-minute sessions covering movement, nutrition, sleep, screen time, stress, and harm prevention for adolescents aged 13-18.

This project was created as part of an assignment requiring students to:
- Identify a real, documented community health need- Design a program responding to that need
- Create a promotional portfolio (in this case, a website) demonstrating understanding of:
  - Physical activity guidelines (including sleep and screen time)
  - Barriers to physical activity and strategies to overcome them  - Benefits of physical activity
  - Dimensions of health and their influence on program design

## Website Structure

The site consists of multiple interconnected pages:
- **index.html**: Landing page with hero section, stats, health issue analysis, why Wyndham focus, and program comparison
- **program.html**: Details how the Rover works, its four pillars (Movement Boosters, Nutrition Skills, Behaviour Change Lessons, Youth Leadership Pathway), session flow, and health areas covered
- **guidelines.html**: Covers physical activity guidelines, dimensions of health, barriers with Rover responses, and benefits- **goals.html**: Outlines program goals, evaluation plan, and potential challenges
- **reflection.html**: Explains the design rationale and background on the creator
- **sources.html**: Lists all referenced materials (including one clearly marked joke source)
- **join.html**: Easter egg page (walnut-themed prank with redirect)
- **clownface.html**: Easter egg page triggered by the "disagree" button
- **shared.js**: Contains navigation, footer, and all interactive easter eggs
- **style.css**: Shared stylesheet defining the dark theme with orange (#ff4500) accents

## Key Features

- **Responsive design**: Works across mobile and desktop devices
- **Interactive elements**: 
  - Konami code triggers clown emoji rain
  - Triple-clicking headers makes the van drive across the screen
  - Typing "rover" anywhere activates the van animation
  - Clicking the van emoji 10x triggers a spin animation
  - Alt+clicking statistics triggers a counting animation  - Mobile users get an idle reminder after 30 seconds
- **Thoughtful content**: 
  - Addresses documented Wyndham health disparities
  - Incorporates Australian physical activity guidelines
  - References real Wyndham health organizations and reports
  - Includes barrier analysis with program-specific solutions
  - Highlights benefits across multiple health dimensions

## Important Notes

- This is a **conceptual proposal** created for an assignment only
- The WynHealth Rover does not exist as an actual operating program
- All health data and references are based on publicly available Wyndham health reports and Australian government guidelines
- The project intentionally uses a modest, student-appropriate design aesthetic (avoiding overly polished "startup" styling)
- Hosted via Cloudflare Pages with manual deployment on pushes to the `main` branch
